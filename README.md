# _Would you rather?_

##### A application made for learning purposes only. It is a clone of rrrather.com. This application practices the use of adding pictures on the page with out reloading. 

#### By _**Alyssa Horrocks and Kendra Ash**_

## Description

_Users:_
* _Can become a user._
* _Can add a would you rather question._
* _Keeps track of the number of votes for each question._
* _Restricts a user from answering a question more than once._


_View App:_ https://github.com/kendraash/would_you_rahter.git

## Database Setup/ Installation

_In rake:_<br>
_db:setup, (The seeded data will be loaded with db:setup.)_<br>
_all database tables listed in schema.rb_
<br>

_Webserver testing:_
* _Install required gems_
* _$ bundle install_
* _rake db:test:prepare_
* _$ rails server_
* _Navigate to localhost:3000_
